### วิธีเข้า
- เริ่มสร้างไฟล์
- กดเข้าไฟล์ที่สร้าง
- พิม ``cmd`` ในแถบค้นหา

```bash
git clone https://github.com/WasitpolKuekkong/ClickNext-Intern-test.git
```
เมื่อติดตั้งเสร็จ

```bash
cd ClickNext-Intern-test
npm i

```

รัน

```bash
npm run dev

```
เปิด

[เข้าหน้าเว็บ](http://localhost:3000/)
